# 人生 online 游戏包

> 一台像素风掌机，装着几个关于人类日常的小游戏。

---

## 简介

「人生 online 游戏包」是一款复古像素风 App，外观模拟经典掌机，内置多个轻量小游戏/工具模块，面向在职场和日常生活中需要一点喘息空间的用户。

全局采用像素字体（Zpix）、Game Boy 配色（#0f380f / #9bbc0f），所有交互都通过掌机按键（十字键 / SELECT / START）或触屏操作完成。

---

## 功能模块

### 职场副本
模拟 RPG 任务系统的待办清单。完成任务获得经验值，积累升级。支持每日任务记录、历史查看。

### 班言班语生成器
输入一句话，选择对象（领导 / 同事 / 甲方）和风格（体面营业 / 温和扎心 / 阴阳有度），AI 一键生成职场回复。由后端 API 驱动。

### 储蓄副本
设定存钱目标和奖励，追踪当前余额与进度。达成目标时解锁奖励弹窗。

### 人生彩蛋
随机抽取一张「人生彩蛋」卡片，显示一句话语录。每张只能抽一次，全部抽完后重置。

### 情绪存档
记录当下情绪状态（7 种预设情绪 + 自定义备注），自动计算出生至今的天数（DAY-X-of-EARTH-ONLINE），支持历史存档查看。

## 本地运行

1. 安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)（App 开发版）
2. 克隆或下载项目，用 HBuilderX 打开项目根目录
3. 运行 → 运行到手机或模拟器

班言班语功能需要网络连接访问 AI 接口，其余功能均完全离线可用。

## 设计规范

- 字体：`'Zpix', monospace`
- 主色：`#0f380f`（深绿）/ `#306230`（中绿）/ `#9bbc0f`（亮绿）
- 所有界面通过 `bootPhase` 状态字符串切换
- 像素风面板阴影：`2px 2px 0 #306230`
